# ramda-destruct

Cleans up required or imported [ramda](http://ramdajs.com) functions when
using destructing assignment.

## install

```sh
npm install -g ramda-destruct
```

## usage

### file as argument

```sh
ramda-destruct path/to/file.js
```

### stdin

```sh
cat path/to/file.js | ramda-destruct
```
